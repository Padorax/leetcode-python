# Hand of Straights

Alice has some number of cards and she wants to rearrange the cards into groups so that each group is of size groupSize, and consists of groupSize consecutive cards.

Given an integer array hand where hand[i] is the value written on the ith card and an integer groupSize, return true if she can rearrange the cards, or false otherwise.


Example 1:
Input: hand = [1,2,3,6,2,3,4,7,8], groupSize = 3
Output: true
Explanation: Alice's hand can be rearranged as [1,2,3],[2,3,4],[6,7,8]

Example 2:
Input: hand = [1,2,3,4,5], groupSize = 4
Output: false
Explanation: Alice's hand can not be rearranged into groups of 4.

Example 3:
Input: hand = [1,2,3,4,5,6], groupSize = 2, True


### Solution
Greedy, hashMap
1. 计数：统计每张牌出现的次数（使用 Counter 或 TreeMap）。
2. 排序：将牌面从小到大排序，这样我们就能按顺序找到“当前最小”的牌。
3. 消耗：
    拿出当前还没被用掉的最小牌 card。
    尝试以此为起点，找齐 card + 1, card + 2, ..., card + groupSize - 1。
    如果中间断了（某张牌不够用），直接返回 False。
    如果找齐了，对应牌的计数减 1，继续找下一个最小牌。
```python
from collections import Counter
class Solution:
    def isNStraightHand(self, hand: List[int], groupSize: int) -> bool:
        if len(hand) % groupSize != 0: return False
        counter = Counter(hand)
        hand = sorted(hand)
        for num in hand:
            if counter[num] < 1: continue
            for j in range(groupSize):
                if counter[num+j] <= 0:
                    return False
                counter[num+j] -= 1
        return True
```
or better
我们不是一张一张去试，而是一批一批去试。如果我有 3 张 1，那么我必须同时消耗 3 张 2 和 3 张 3。这种“批量消耗”的思想让 $O(N \log N)$ 的排序开销变得非常值得。
```python
from collections import Counter

class Solution:
    def isNStraightHand(self, hand: List[int], groupSize: int) -> bool:
        if len(hand) % groupSize != 0:
            return False
            
        count = Counter(hand)
        # 获取所有去重并排好序的牌面
        sorted_keys = sorted(count.keys())
        
        for card in sorted_keys:
            if count[card] > 0:
                # 记录当前最小牌需要作为多少个顺子的起点
                needed = count[card]
                # 尝试构建以 card 为起点的顺子
                for i in range(groupSize):
                    target = card + i
                    if count[target] < needed:
                        return False
                    count[target] -= needed
                    
        return True
```
