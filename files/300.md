### Solution 1
DP, 我们定义dp[i]表示以nums[i]结尾的最长递增子序列的长度，初始时dp[i] = 1, 答案为dp[i]的最大值。
for dp[i], iterate 0<=j<i, if nums[j] < nums[i], then dp[i] = max(dp[i], dp[j] + 1)

Time complexity: O(N^2)</br>
Space complexity: O(N)

```python
def lengthOfLIS(nums: List[int]) -> int:
        n = len(nums)
        if n < 2: return n
        
        dp = [1] * n
        for i in range(n):
            for j in range(i):
                if nums[j] < nums[i]:
                    dp[i] = max(dp[i], dp[j] + 1)
        return max(dp)
```

### Solution 2 Greedy + Binary
维护一个数组tails: tails[k] = 长度为 k+1 的递增子序列的最小结尾 (我们不存序列，只存“最优结尾”)
Ex: nums = [10,9,2,5,3,7,101,18]
tails
`
[10]
[9]
[2]
[2,5]
[2,3]
[2,3,7]
[2,3,7,101]
[2,3,7,18]
`
👉 结尾越小，未来越容易接更大的数.为什么可以替换tails?因为只关心： 通常长度下，谁的结尾更小
```python
import bisect

def lengthOfLIS(nums):
    tails = []

    for num in nums:
        # 找到第一个 >= num 的位置
        idx = bisect.bisect_left(tails, num)

        if idx == len(tails):
            tails.append(num)
        else:
            tails[idx] = num

    return len(tails)
```
or with custom bisect:
```python
class Solution:
    def lengthOfLIS(self, nums: List[int]) -> int:
        tails = []
        for num in nums:
            pos = self.findInsertion(tails, num)
            if pos == len(tails):
                tails.append(num)
            else:
                tails[pos] = num
        return len(tails)
    
    def findInsertion(self, nums, target):
        l, r = 0, len(nums) #<- len(nums) so for [1, 4] and 6 return idx=2
        while l < r:
            mid = (l + r) // 2
            if nums[mid] < target:
                l = mid + 1
            else:
                r = mid
        return l
```
time: O(nlogn), space: O(n)

注意这里的tail并不是最终的结果， e.g., [2, 3, 1, 4]
tail: [], [2], [2, 3], [1, 2], [1, 2, 4]
长度3是对的，但是正确的序列时[2, 3, 4]而不是[1, 2, 4]

假设真实L = [a1, a2, a3, ..., ak],我们会保证在某个时刻
tails[0] <= a1  
tails[1] <= a2  
tails[2] <= a3  
...
i.e., LIS能多长， tails就能多长

### Solution 2.2
Refer to [here](https://github.com/doocs/leetcode/blob/main/solution/0300-0399/0300.Longest%20Increasing%20Subsequence/README.md), time complexity: O(NLogN)


```python
class BinaryIndexedTree:
    def __init__(self, n: int):
        self.n = n
        self.c = [0] * (n + 1)

    def update(self, x: int, v: int):
        while x <= self.n:
            self.c[x] = max(self.c[x], v)
            x += x & -x

    def query(self, x: int) -> int:
        mx = 0
        while x:
            mx = max(mx, self.c[x])
            x -= x & -x
        return mx


class Solution:
    def lengthOfLIS(self, nums: List[int]) -> int:
        s = sorted(set(nums))
        m = len(s)
        tree = BinaryIndexedTree(m)
        for x in nums:
            x = bisect_left(s, x) + 1
            t = tree.query(x - 1) + 1
            tree.update(x, t)
        return tree.query(m)
```

### Solution DP (top-down)
Instead of tracking both current index and last included index, we can define dfs(i) as the length of the longest increasing subsequence starting at index i. For each starting position, we look at all positions j > i where nums[j] > nums[i] and take the maximum. This reduces the state to just one dimension.
```python
class Solution:
    def lengthOfLIS(self, nums: List[int]) -> int:
        n = len(nums)
        memo = [-1] * n

        def dfs(i):
            if memo[i] != -1:
                return memo[i]

            LIS = 1
            for j in range(i + 1, n):
                if nums[i] < nums[j]:
                    LIS = max(LIS, 1 + dfs(j))

            memo[i] = LIS
            return LIS

        return max(dfs(i) for i in range(n))
```
time: O(n^2), space: O(n)

### Solution DP + binary search
We maintain an array dp where dp[i] is the smallest ending element of all increasing subsequences of length i + 1. This array stays sorted. For each new element, if it's larger than the last element in dp, it extends the longest subsequence. Otherwise, we use binary search to find the position where it can replace an element, keeping the array optimal for future extensions.

```python
import bisect

def lengthOfLIS(nums):
    # tails[k] = 长度为 k+1 的递增子序列的最小结尾
    tails = []

    for num in nums:
        # 找到第一个 >= num 的位置
        # or binary_search_insertion_pos(num+1)
        idx = bisect.bisect_left(tails, num)

        if idx == len(tails):
            tails.append(num)
        else:
            tails[idx] = num

    return len(tails)
```
