### Solution 1 DP - bottom up

f(n) is the max value using first n+1 element.
```
f(0) = nums[0]
f(1) = max(nums[0], nums[1])
f(n) = max(f(n-1), f(n-2)+nums[i]) -> corresponding to do not take n-th element, or take n-th element

Note f(n) is non-decreasing so return dp[-1]
```
```python
class Solution:
    def rob(self, nums: List[int]) -> int:
        if len(nums) == 1:
            return nums[0]
        dp = [0] * len(nums)
        dp[0] = nums[0]
        dp[1] = max(nums[0], nums[1])
        for i in range(2, len(nums)):
            dp[i] = max(dp[i-1], dp[i-2]+ nums[i])
        return dp[-1]
```

With space optimization
```python
def rob(nums: List[int]) -> int:
    last, now = 0, 0
    for n in nums:
        last, now = now, max(last+n, now)
    return now
```

### Solution 2 recursion (less ideal)
```python
class Solution:
    def rob(self, nums: List[int]) -> int:

        def dfs(i):
            if i >= len(nums):
                return 0
            return max(dfs(i + 1),
                       nums[i] + dfs(i + 2))

        return dfs(0)
```
