# Partition palindrom
Given a string s, partition s such that every substring of the partition is a palindrome. Return all possible palindrome partitioning of s.

Example 1:
Input: s = "aab", Output: [["a","a","b"],["aa","b"]]

Example 2:
Input: s = "a", Output: [["a"]]
 
Constraints:
1 <= s.length <= 16
s contains only lowercase English letters.

### Solution 1 backtracing
```python
class Solution:
    def partition(self, s: str) -> List[List[str]]:
        res = []
        def isPalindrom(s, i, j):
            while i < j:
                if s[i] != s[j]:
                    return False
                i += 1
                j -= 1
            return True

        def helper(idx, cur): # [.,idx],[idx+1, ..)
            if idx == len(s):
                res.append(cur)
                return
            for i in range(idx, len(s)): #careful here len(s) not len(s)-1
                if isPalindrom(s, idx, i): # add palindrom s[idx:i+1], next palindrom starting from i+1
                    helper(i+1, cur+[s[idx:i+1]])
        helper(0, [])
        return res
```
time complexity: O(n*2^n), space: O(n) extra space, O(n*2^n) for the output list

### Solution 2 backtracing (DP)
我们维护一个二维数组 dp，其中 dp[i][j] 表示子串 s[i:j+1] 是否为回文。
* 基础情况 1：如果 i=j，单个字符一定是回文。
* 基础情况 2：如果 s[i]=s[j] 且两个字符相邻（即 j−i=1），它是回文。
状态转移：如果 s[i]=s[j] 且其内部的子串 s[i+1:j-1] 是回文，那么 s[i:j+1] 也是回文。

```python
class Solution:
    def partition(self, s: str) -> List[List[str]]:
        n = len(s)
        res = []
        
        # 1. DP 预处理所有回文子串状态
        # dp[i][j] 代表 s[i...j] 是否为回文
        dp = [[False] * n for _ in range(n)]
        
        # 注意：为了计算 dp[i][j]，我们需要先知道 dp[i+1][j-1]
        # 所以 i 要从大到小遍历，j 要从小到大遍历 (ex: n = 4 aabc
        #  0 1 2 3
        #0 T
        #1   T
        #2     T (F)
        #3       T
        for i in range(n - 1, -1, -1):
            for j in range(i, n):
                # if i == j: dp[i][j] = True omit
                if s[i] == s[j]:
                    # 情况：i 和 j 是同一个字母，或者是相邻字母，或者内部也是回文
                    if j - i <= 2 or dp[i+1][j-1]:
                        dp[i][j] = True

        # 2. 回溯函数
        def backtrack(start, path):
            if start == n:
                res.append(list(path)) # deep copy, or path.copy()
                return
            
            for end in range(start, n):
                # 使用 DP 结果进行 O(1) 判断
                if dp[start][end]:
                    path.append(s[start:end+1])
                    backtrack(end + 1, path)
                    path.pop()

        backtrack(0, [])
        return res
```
