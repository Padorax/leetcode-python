# Task Scheduler

### Solution 1 Math
Ref: https://medium.com/@satyem77/task-scheduler-leetcode-39d579f3440

```python
class Solution:
    def leastInterval(self, tasks: List[str], n: int) -> int:
        cnt = Counter(tasks)
        x = max(cnt.values())
        s = sum(v == x for v in cnt.values())
        return max(len(tasks), (x - 1) * (n + 1) + s)
```
or
```python
class Solution:
    def leastInterval(self, tasks: List[str], n: int) -> int:
        counter = dict()
        for t in tasks:
            counter[t] = counter.get(t, 0) + 1
        maxOccur = max(counter.values())
        numOfMaxOccur = 0
        for t, cnt in counter.items():
            if cnt == maxOccur:
                numOfMaxOccur += 1
        
        return max(len(tasks), (maxOccur - 1) * (n + 1) + numOfMaxOccur)
```

### Solution 2 Max-Heap
核心逻辑：为什么用大顶堆？
为了总时间最短，我们必须优先处理出现次数最多的任务。如果你先处理掉那些只出现 1 次的任务，最后剩下 5 个 A，你将不得不被迫在 A 之间插入大量的 idle（空闲时间）。

 ```python
 class Solution:
    def leastInterval(self, tasks: List[str], n: int) -> int:
        count = Counter(tasks)
        maxHeap = [-cnt for cnt in count.values()]
        heapq.heapify(maxHeap)

        time = 0
        q = deque()  # pairs of [-cnt, end of idleTime]
        while maxHeap or q:
            time += 1
            
            if not maxHeap:
                # 优化：如果堆空了，说明此时只能等待。
                # 直接把时间跳到队列中第一个任务解封的时间，避免 time += 1 一步步空跳。
                time = q[0][1]
            else:
                # 拿出执行次数最多的任务，执行一次
                cnt = 1 + heapq.heappop(maxHeap) # 因为是负数，+1 代表次数减 1
                if cnt:
                    # 如果还有剩余次数，放入冷宫 q，记录它在 time + n 时解封
                    q.append([cnt, time + n])

            # 检查冷宫里有没有任务到期了
            if q and q[0][1] == time:
                # 到期了，放回大顶堆，重新参与竞争
                heapq.heappush(maxHeap, q.popleft()[0])
        return time
 ```
 假设任务是 [A, A, A, B, B, B]，n = 2：
 Time 1: 执行 A。A 剩 2 个，进入冷宫，解禁时间为 1 + 2 = 3。
 Time 2: 执行 B。B 剩 2 个，进入冷宫，解禁时间为 2 + 2 = 4。
 Time 3: 此时 A 解禁，从冷宫回堆。立刻执行 A。A 剩 1 个，解禁时间为 3 + 2 = 5。...以此类推，直到所有任务清零。
