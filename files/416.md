# Partition Equal Subset Sum

You are given an array of positive integers nums.

Return true if you can partition the array into two subsets, subset1 and subset2 where sum(subset1) == sum(subset2). Otherwise, return false.
Example 1:

Input: nums = [1,2,3,4]

Output: true
Explanation: The array can be partitioned as [1, 4] and [2, 3].

Example 2:

Input: nums = [1,2,3,4,5]

Output: false

### Solution brutal force
Can get all possible subsets of nums, see if any with sum == sum(nums)/2

### Solution DP (0/1 backpack)
转成 0/1 背包问题, 我们要做的是：每个数只能用一次（0/1）是否能凑出 target = S/2
定义：dp[i][j] = 前 i 个数，能不能凑出和 j
转移：dp[i][j] = dp[i-1][j]  # 不选
        or dp[i-1][j - nums[i]]  # 选
优化成一维 DP （因为只用到上一行）


```python
class Solution:
    def canPartition(self, nums: List[int]) -> bool:
        total = sum(nums)
        # 奇数直接不可能
        if total % 2 != 0:
            return False

        target = total // 2
        # dp[i][j]: can achieve sum j with first i elements (range)
        # dp[i][j] = dp[i-1][j] or dp[i-1][j-nums[i]]

        # 和为0一定可以 （一个都不选）
        dp = [True] + [False]*target
        for num in nums:
            # 倒序遍历（关键！）
            # 如果用2d matrix 可以正序,这里我们需要dp[i-1]（上一行），所以需要在更新dp[j]时dp[j-nums[i]]仍是上一行的值
            # also note 如果dp[i][j] = dp[i][j-x]那么只能从左往右
            for j in range(target, num-1, -1):
                dp[j] = dp[j] or dp[j-num]
            
            if dp[target]: return True
        return dp[target]
```
time complexity: O(n*target)
space: O(target)
