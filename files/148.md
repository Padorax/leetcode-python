### Solution
我们可以用归并排序的思想来解决。首先，我们利用快慢指针找到链表的中点，将链表从中点处断开，形成两个独立的子链表l1, l2。然后，我们递归地对l1, l2进行排序，最后将l1, l2合并为一个有序链表。
time complexity: O(nlogn), space: O(logn)
```python
class Solution:
    def sortList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        if not head or not head.next:
            return head
        slow, fast = head, head.next
        while fast and fast.next:
            slow = slow.next
            fast = fast.next.next
        
        l1, l2 = head, slow.next
        slow.next = None
        l1, l2 = self.sortList(l1), self.sortList(l2)

        dummy = ListNode(None)
        tail = dummy
        while l1 and l2:
            if l1.val <= l2.val:
                tail.next = l1
                l1 = l1.next
            else:
                tail.next = l2
                l2 = l2.next
            tail = tail.next
        tail.next = l1 or l2
        return dummy.next
```
