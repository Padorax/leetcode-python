### Solution
```python
class Solution:
    def reverseBits(self, n: int) -> int:
        res = 0
        i = 0
        while n:
            rem = n & 0x1
            n = n >> 1
            res = (res << 1) + rem
            i += 1
        if i < 32:
            res = res << (32-i)
        return res
```
