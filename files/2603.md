# Collect Coins in a Tree
There exists an undirected and unrooted tree with n nodes indexed from 0 to n - 1. You are given an integer n and a 2D integer array edges of length n - 1, where edges[i] = [ai, bi] indicates that there is an edge between nodes ai and bi in the tree. You are also given an array coins of size n where coins[i] can be either 0 or 1, where 1 indicates the presence of a coin in the vertex i.

Initially, you choose to start at any vertex in the tree. Then, you can perform the following operations any number of times: 

Collect all the coins that are at a distance of at most 2 from the current vertex, or
Move to any adjacent vertex in the tree.
Find the minimum number of edges you need to go through to collect all the coins and go back to the initial vertex.

Note that if you pass an edge several times, you need to count it into the answer several times.


Example 1:
Input: coins = [1,0,0,0,0,1], edges = [[0,1],[1,2],[2,3],[3,4],[4,5]]
Output: 2
Explanation: Start at vertex 2, collect the coin at vertex 0, move to vertex 3, collect the coin at vertex 5 then move back to vertex 2.

Example 2:
Input: coins = [0,0,0,1,1,0,0,1], edges = [[0,1],[0,2],[1,3],[1,4],[2,5],[5,6],[5,7]]
Output: 2
Explanation: Start at vertex 0, collect the coins at vertices 4 and 3, move to vertex 2,  collect the coin at vertex 7, then move back to vertex 0.

### Solution
我们需要在树中移动，收集所有金币，并回到起点。关键点在于：我们不需要走到每一个节点，只需要走到覆盖所有金币的“核心树”区域即可。解题思路：
- 修剪无用叶子（第一次拓扑排序）：那些不含金币的叶子节点是完全没必要访问的。我们可以反复删除这类节点，直到所有叶子节点都包含金币。
- 修剪必经路径（第二次拓扑排序）：由于我们可以在距离 2 的范围内收集金币，这意味着我们也不必走进金币节点的“深处”（距离叶子节点 2 层以内的部分）。因此，我们再进行两次拓扑排序，修剪掉叶子节点及其父节点，剩下的就是必须经过的核心路径。
- 计算结果：剩下的边数乘以 2 即为答案（因为是往返路径）。如果修剪后节点数 $\le 1$，则答案为 0。
---
这道题的解法分为三步“剪枝”：

第一步：剪掉“没钱的枯枝”。
如果一个叶子节点（度为 1）没有金币，那它和连接它的那条边完全没用，直接删掉。删掉后可能产生新的“没钱叶子”，继续删（拓扑排序）。

第二步：剪掉“金币叶子”的第一层。
现在剩下的树，叶子节点一定都有金币。但规则说你可以距离 2 收集，所以你不需要走到叶子，只要走到它的父亲就能收割它。所以把所有叶子删掉。

第三步：再剪一层。
同理，你其实只需要走到叶子的爷爷那里，就能收割掉刚才那个叶子。所以再删掉一轮当前的叶子。

最后剩下的树，就是你必须走过的路径。(起点不重要)
```python
from collections import deque

class Solution:
    def collectTheCoins(self, coins: list[int], edges: list[list[int]]) -> int:
        n = len(coins)
        if n <= 1: return 0
        
        # 1. 初始化邻接表和度数数组
        adj = [set() for _ in range(n)]
        degree = [0] * n
        for u, v in edges:
            adj[u].add(v)
            adj[v].add(u)
            degree[u] += 1
            degree[v] += 1
            
        # 2. 第一次剪枝：拓扑排序移除所有【无金币】的叶子
        # 队列里只放：度为 1 且 没钱的点
        queue = deque([i for i in range(n) if degree[i] == 1 and coins[i] == 0])
        
        deleted_edges = 0
        while queue:
            u = queue.popleft()
            for v in adj[u]:
                adj[v].remove(u)
                degree[v] -= 1
                deleted_edges += 1
                if degree[v] == 1 and coins[v] == 0:
                    queue.append(v)
            # 清空当前点的邻接信息，防止重复处理
            adj[u].clear()
            degree[u] = 0

        # 3. 第二次剪枝：剥掉两层【金币叶子】
        # 此时剩下的叶子节点一定都有金币。我们要往内缩 2 层。
        leaves = deque([i for i in range(n) if degree[i] == 1])
        
        for _ in range(2):
            # 必须按层处理（Level-order traversal）
            for _ in range(len(leaves)):
                u = leaves.popleft()
                for v in adj[u]:
                    adj[v].remove(u)
                    degree[v] -= 1
                    deleted_edges += 1
                    if degree[v] == 1: # when v becomes a leaf
                        leaves.append(v)
                adj[u].clear()
                degree[u] = 0
        
        # 总边数 = (n-1)，往返路径 = (n-1) * 2
        # 结果 = 总路径 - 被删掉的边数 * 2
        res = (n - 1 - deleted_edges) * 2
        return res
        #return max(0, res)
```
