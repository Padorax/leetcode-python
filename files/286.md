# Walls and Gates / Treasure and islands
You are given a m×n 2D grid initialized with these three possible values:

-1 - A water cell that can not be traversed.
0 - A treasure chest.
INF - A land cell that can be traversed. We use the integer 2^31 - 1 = 2147483647 to represent INF.

Fill each land cell with the distance to its nearest treasure chest. If a land cell cannot reach a treasure chest then the value should remain INF.

Assume the grid can only be traversed up, down, left, or right.
Modify the grid in-place.

Input: [
  [2147483647,-1,0,2147483647],
  [2147483647,2147483647,2147483647,-1],
  [2147483647,-1,2147483647,-1],
  [0,-1,2147483647,2147483647]
]

Output: [
  [3,-1,0,1],
  [2,2,1,-1],
  [1,-1,2,-1],
  [0,-1,3,4]
]

Input: [
  [0,-1],
  [2147483647,2147483647]
]

Output: [
  [0,-1],
  [1,2]
]

### Solution BFS - TLE
```python
from collections import deque

class Solution:
    def islandsAndTreasure(self, grid: List[List[int]]) -> None:
        m, n = len(grid), len(grid[0])
        dirs = [(-1, 0), (1, 0), (0, 1), (0, -1)]
        
        def bfs(i, j):
            # 每一个格子出发，都要找离它最近的 0
            q = deque([(i, j, 0)]) # 存入 (行, 列, 当前步数)
            visited = {(i, j)}
            while q:
                r, c, step = q.popleft()
                for dr, dc in dirs:
                    nr, nc = r + dr, c + dc
                    # 1. 边界检查
                    if 0 <= nr < m and 0 <= nc < n and grid[nr][nc] != -1:
                        # 2. 如果碰到宝藏，这就是最短路径，直接更新并返回
                        if grid[nr][nc] == 0:
                            grid[i][j] = step + 1
                            return
                        # 3. 没访问过且是空地，继续搜
                        if (nr, nc) not in visited:
                            visited.add((nr, nc))
                            q.append((nr, nc, step + 1))
        
        for i in range(m):
            for j in range(n):
                # 只有空地才需要寻找宝藏
                if grid[i][j] == 2147483647:
                    bfs(i, j)
```

### Solution Multi-start BFS
```python
class Solution:
    def islandsAndTreasure(self, grid: List[List[int]]) -> None:
        m, n = len(grid), len(grid[0])
        dirs = [(-1, 0), (1, 0), (0, 1), (0, -1)]
        # 从宝藏出发，向四周扩散
        q = [(i, j) for i in range(m) for j in range(n) if grid[i][j] == 0]

        step = 1
        while q:
            for _ in range(len(q)):
                r, c = q.pop(0)
                for dr, dc in dirs:
                    nr, nc = r+dr, c+dc
                    # 如果相邻格子在范围内，且是待填充的空地 (2147483647)
                    if 0 <= nr < m and 0 <= nc < n and grid[nr][nc] == 2147483647:
                        grid[nr][nc] = step
                        # 将这个填好的格子加入队列，继续扩散
                        q.append((nr, nc))
            step += 1
```

```python
from collections import deque

class Solution:
    def islandsAndTreasure(self, grid: List[List[int]]) -> None:
        if not grid: return
        m, n = len(grid), len(grid[0])
        q = deque()
        
        # 1. 将所有宝藏 (0) 的坐标加入队列，作为多源 BFS 的起点
        for r in range(m):
            for c in range(n):
                if grid[r][c] == 0:
                    q.append((r, c))
        
        # 2. 从宝藏出发，向四周扩散
        while q:
            r, c = q.popleft()
            
            for dr, dc in [(-1, 0), (1, 0), (0, 1), (0, -1)]:
                nr, nc = r + dr, c + dc
                
                # 如果相邻格子在范围内，且是待填充的空地 (2147483647)
                if 0 <= nr < m and 0 <= nc < n and grid[nr][nc] == 2147483647:
                    # 距离 = 当前格子的距离 + 1
                    grid[nr][nc] = grid[r][c] + 1
                    # 将这个填好的格子加入队列，继续扩散
                    q.append((nr, nc))
```
为什么你之前的代码逻辑行不通？
* 性能黑洞：你的代码对每个房间都做一次完整的 BFS。如果棋盘很大且宝藏很少，重复搜索的量极其惊人。
* 多源 BFS 的妙处：想象一下，你在每个宝藏点放一滴“墨水”。墨水同时向四周扩散，当某滴墨水染到一个空地时，那个深度一定就是该空地距离最近宝藏的最短距离。空
* 间优化：这种做法不需要 visited 集合，因为 grid[nr][nc] == 2147483647 本身就起到了“未访问过”的判断作用。一旦被填了数字，它就不再是 2147483647，也就不会被重复处理。

复杂度分析时间复杂度：O(m*n)。每个格子最多入队出队一次。
空间复杂度：O(m*n)。最坏情况下队列里可能存下所有的格子。💡 
极客笔记：多源 BFS 的套路 - 凡是遇到**“求棋盘上所有点到最近目标点的距离”这种题，永远不要从“点”出发，要从“目标”出发**：
1. 找出所有目标点（宝藏、门、火源等）。
2. 一次性全部塞进 deque。
3. 标准 BFS 扩散。
这种思路在 01 Matrix 或者 Rotting Oranges（腐烂的橘子）中是一模一样的。
