# Permutation in String

Given two strings s1 and s2, return true if s2 contains a permutation of s1, or false otherwise.

In other words, return true if one of s1's permutations is the substring of s2.


Example 1:

Input: s1 = "ab", s2 = "eidbaooo"
Output: true
Explanation: s2 contains one permutation of s1 ("ba").
Example 2:

Input: s1 = "ab", s2 = "eidboaoo"
Output: false
 

Constraints:

1 <= s1.length, s2.length <= 104
s1 and s2 consist of lowercase English letters.


### Solution
```python
class Solution:
    def checkInclusion(self, s1: str, s2: str) -> bool:
        if len(s1) > len(s2):
            return False

        # Frequency count arrays for s1 and the current window in s2
        s1_count = [0] * 26
        s2_count = [0] * 26

        # Initialize the frequency count for s1 and the first window in s2
        for i in range(len(s1)):
            s1_count[ord(s1[i]) - ord('a')] += 1
            s2_count[ord(s2[i]) - ord('a')] += 1

        # Slide the window over s2
        for i in range(len(s2) - len(s1)):
            if s1_count == s2_count:
                return True
            # Remove the leftmost character of the current window
            s2_count[ord(s2[i]) - ord('a')] -= 1
            # Add the next character to the window
            s2_count[ord(s2[i + len(s1)]) - ord('a')] += 1

        # Check the last window
        return s1_count == s2_count
```
### Solution optimized
Instead of checking s1_count and updated s2_count over and over again. we track change:
通常的解法是每次滑动窗口后执行 if s1arr == s2arr:，这在 Python 中是 O(26)。 优化后的思路是：只在字符进入或离开窗口时，观察它对“匹配总数”的贡献。

进入窗口 (Right)：

如果增加一个字符后，窗口内该字符的数量刚好等于目标数量 → matches += 1。

如果增加前该字符数量刚好等于目标数量，增加后多了一个 → matches -= 1（原本匹配，现在不匹配了）。

离开窗口 (Left)：

如果减少一个字符后，窗口内该字符的数量刚好等于目标数量 → matches += 1。

如果减少前该字符数量刚好等于目标数量，减少后少了一个 → matches -= 1（原本匹配，现在不匹配了）。
```python
class Solution:
    def checkInclusion(self, s1: str, s2: str) -> bool:
        n1, n2 = len(s1), len(s2)
        if n1 > n2:
            return False

        s1_counts = [0] * 26
        s2_counts = [0] * 26
        
        # 初始化前 n1 个字符的频率
        for i in range(n1):
            s1_counts[ord(s1[i]) - ord('a')] += 1
            s2_counts[ord(s2[i]) - ord('a')] += 1

        # 初始化 matches 数（有多少个字母的频率是匹配的）
        matches = 0
        for i in range(26):
            if s1_counts[i] == s2_counts[i]:
                matches += 1

        # 开始滑动窗口
        # i 是窗口左端点要移除的索引
        for i in range(n2 - n1):
            if matches == 26:
                return True
            
            # 右侧字符进入窗口
            r_idx = ord(s2[i + n1]) - ord('a')
            s2_counts[r_idx] += 1
            if s2_counts[r_idx] == s1_counts[r_idx]:
                matches += 1
            elif s2_counts[r_idx] == s1_counts[r_idx] + 1:
                matches -= 1

            # 左侧字符离开窗口
            l_idx = ord(s2[i]) - ord('a')
            s2_counts[l_idx] -= 1
            if s2_counts[l_idx] == s1_counts[l_idx]:
                matches += 1
            elif s2_counts[l_idx] == s1_counts[l_idx] - 1:
                matches -= 1

        return matches == 26
```
